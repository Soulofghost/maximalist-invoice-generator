# Maximalist Invoice Generator 🎨💥

A bold, high-contrast, "more is more" web-based invoice generator built with Vanilla HTML, CSS, and JavaScript. Say goodbye to boring corporate invoices and hello to neon colors, sharp shadows, and Neo-Brutalism aesthetics!

## Features ✨

- **Loud & Proud Design**: Utilizes vibrant colors (Lime Green, Hot Pink, Electric Blue) and bold typography (`Syne` and `Space Grotesk`).
- **Real-Time Preview**: The invoice preview updates instantly as you type out sender, client, and item details.
- **Dynamic Item Management**: Add or remove line items on the fly with automatic subtotal, tax, and grand total calculations.
- **Print-Ready / Export to PDF**: A dedicated `@media print` stylesheet strips away the chaotic UI components, ensuring your final exported PDF or printed invoice looks professional and clean.

## How to Use 🚀

Since this is built entirely with vanilla web technologies, there's no build step required!

1. Clone the repository:
   ```bash
   git clone https://github.com/Soulofghost/maximalist-invoice-generator.git
   ```
2. Open the folder and double-click `index.html` to open it in your web browser.
3. Fill out the form fields with your details, the client's details, and your invoice line items.
4. Click the **PRINT / SAVE AS PDF** button at the bottom of the page to generate your final invoice.

## Tech Stack 🛠️

- HTML5
- CSS3 (Vanilla, CSS Variables, Flexbox, Neo-Brutalism styling)
- JavaScript (Vanilla DOM manipulation)
